# Next.js Resume

A single-page resume built with Next.js (App Router) and Tailwind CSS.

## How to Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features Implemented

- [x] Next.js project scaffolded with Tailwind CSS
- [x] Resume content in `src/app/page.tsx` 
- [x] All sections present: Summary, Education, Experience, Projects, Certifications, Skills
- [x] Real content — no placeholder text
- [x] Semantic HTML elements (`<header>`, `<main>`, `<section>`, `<footer>`)
- [x] All styling done with Tailwind CSS utility classes via `className`
- [x] Clean, readable layout at desktop viewport
- [x] No console errors
- [x] Default boilerplate cleaned up

## Known Issues / Limitations

- Layout is optimized for desktop viewport only
